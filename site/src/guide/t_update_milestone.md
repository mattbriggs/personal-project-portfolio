# Advance a Milestone to Done

Mark a milestone as done when the outcome it describes has been achieved.

Milestones do not advance automatically. You mark them done when you decide the outcome is complete. This is a deliberate checkpoint—before marking a milestone done, confirm that the outcome truly meets the standard you defined when you created it.

1.  Click the **Milestones** tab.

2.  Select the project from the dropdown.

3.  Select the milestone in the list.

4.  Click **Edit Milestone**.

5.  Change the **Status** field to **Done**.

    Portfolio Manager records the completion date automatically.

6.  Click **Save**.

    The milestone's status updates in the list. The project's milestone score component increases, which may change the project's overall health score and traffic-light status on the Dashboard.


After marking a milestone done, check whether any new milestones should be promoted from Backlog to Planned to maintain forward momentum.
