# Create a Project

Add a new project to your portfolio and configure its metadata.

Create a project for each distinct commitment in your portfolio. A commitment qualifies as a project if it has a meaningful end state, benefits from regular sessions, and would suffer from being deprioritized indefinitely.

1.  Click the **Projects** tab.

2.  Click **New Project**.

    The Project dialog opens.

3.  In the **Project** field, enter a short, recognizable name.

    Use a name you will recognize at a glance on the Dashboard. For example: "Novel Draft," "Home Studio Album," or "AWS Certification."

4.  In the **Description** field, enter a one- or two-sentence summary of the project's purpose and desired outcome.

5.  Set the **Status** field.

    -   Select **Active** if you intend to schedule sessions for this project this week.
    -   Select **Backlog** if you are capturing the project but are not ready to start yet.
6.  Set the **Priority** field to a value from 1 \(highest\) to 5 \(lowest\).

    Priority is for your reference. It does not affect scoring or display order.

7.  \(Optional\) Enter a **Start** date and an **End** date in YYYY-MM-DD format, or click **Today** to fill in today's date.

8.  Click **Save**.

    The project appears in the Projects list. If status is Active, it also appears on the Dashboard.


After creating a project, add milestones to define its key outcomes. See [Create a Milestone](t_create_milestone.md).
