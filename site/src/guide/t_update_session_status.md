# Update a Session's Status

Mark a session as Done after you complete it, or Cancelled if you abandon it. Status updates drive the project health score on the Dashboard.

1.  Click the **Sessions** tab and navigate to the week containing the session.

2.  Select the session in the table.

3.  Click **Mark Done** to complete the session in one step.

    Alternatively, click **Edit Session** to open the full dialog and change the status field manually. Use the Edit dialog when you also want to add notes about what you accomplished.

    The session status changes to Done. Portfolio Manager records the completion timestamp. The Dashboard score for the project updates immediately.


The session is recorded as complete for the current week. Its duration counts toward the Done total in the weekly budget bar.

## To Cancel a Session

If you decide not to complete a session, select it in the Sessions table, click **Edit Session**, change the status to **Cancelled**, and save. Cancelled sessions do not count toward the weekly budget or the project score. You can delete cancelled sessions to keep the list tidy.

**Tip:** If you cannot complete a session this week but still want to do the work, edit the session date to move it to a future week rather than cancelling it. This preserves the intent and keeps the session in your backlog.
