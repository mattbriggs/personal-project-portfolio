# Archive a Project

Archive a project when it is complete, discontinued, or no longer active. Archiving preserves the project's history while removing it from the active Dashboard.

Archive a project in two situations: when all milestones are achieved and the project is complete, or when you have decided to discontinue the project. Archived projects are read-only—you can review their history, but you cannot add sessions or milestones to them.

If you want to pause a project temporarily without archiving it, move it to Backlog status instead.

1.  Click the **Projects** tab.

2.  Select the project you want to archive.

3.  Click **Archive Project**.

    A confirmation prompt appears asking you to confirm the archive action.

4.  Click **Confirm**.

    The project status changes to Archive. It disappears from the Dashboard and from the active project filter. It remains visible when you set the Projects tab filter to **Archive** or **All**.


If you archived a project by mistake, you can restore it: open the Projects tab, set the status filter to **Archive**, select the project, click **Edit Project**, change the status back to **Active** or **Backlog**, and save.
