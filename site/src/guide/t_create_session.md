# Create a Session

Schedule a work session for a project by specifying its date, duration, and purpose.

Sessions are your primary planning tool. Create sessions at the start of the week to commit to specific blocks of work, or create them on the day you plan to work if you prefer a more spontaneous workflow.

1.  Click the **Sessions** tab.

2.  Verify that the week displayed in the week selector matches the week you are planning for. Use the arrow buttons to navigate to a different week if needed.

3.  Click **New Session**.

    The Session dialog opens.

4.  In the **Project** dropdown, select the project this session belongs to.

5.  \(Optional\) In the **Milestone** dropdown, select the milestone this session will advance.

    Only milestones for the selected project are shown. You can change or clear this field later.

6.  In the **Session** field, enter a brief label for what this session will accomplish.

    Keep this to one phrase: "Write act two outline," "Record guitar tracks," "Study AWS IAM chapter." It appears in the Sessions table and the Dashboard.

7.  In the **Date** field, enter the scheduled date in YYYY-MM-DD format, or click **Today** to fill in today's date.

8.  Set the **Status** field.

    -   Select **Planned** if this session is committed for this week and should count toward the weekly budget.
    -   Select **Backlog** if you are capturing the session idea without committing to a specific week.
9.  In the **Min** spinbox, set the session duration in minutes.

    The default is set in **Settings**. Valid values are 15 to 480.

10. \(Optional\) In the **Description** field, add any notes or context for the session—references to read, questions to explore, or specifics about what to accomplish.

11. Click **Save**.

    The session appears in the Sessions table. If status is Planned, the weekly budget bar updates to reflect the added duration.
